An internet socket address, either IPv4 or IPv6.

Internet socket addresses consist of an IP address, a 16-bit port number, as well as possibly some version-dependent additional information. See SocketAddrV4â€™s and SocketAddrV6â€™s respective documentation for more details.

Â§Portability

SocketAddr is intended to be a portable representation of socket addresses and is likely not the same as the internal socket address type used by the target operating systemâ€™s API. Like all repr(Rust) structs, however, its exact layout remains undefined and should not be relied upon between builds.
