Trait which permits the allowed types to be used with VaListImpl::arg.

ยงSafety

This trait must only be implemented for types that C passes as varargs without implicit promotion.

In C varargs, integers smaller than c_int and floats smaller than c_double are implicitly promoted to c_int and c_double respectively. Implementing this trait for types that are subject to this promotion rule is invalid.
