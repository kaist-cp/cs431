pub fn _mm_extracti_si64<const LEN: i32, const IDX: i32>(x: __m128i) -> __m128i
Expand description

Extracts the specified bits from the lower 64 bits of the 128-bit integer vector operand at the index idx and of the length len.

idx specifies the index of the LSB. len specifies the number of bits to extract. If length and index are both zero, bits [63:0] of parameter x are extracted. It is a compile-time error for len + idx to be greater than 64 or for len to be zero and idx to be non-zero.

Returns a 128-bit integer vector whose lower 64 bits contain the extracted bits.
