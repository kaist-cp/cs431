Expand description

256-bit wide set of 16 u16 types, x86-specific

This type is the same as the __m256bh type defined by Intel, representing a 256-bit SIMD register which internally is consisted of 16 packed u16 instances. Its purpose is for bf16 related intrinsic implementations.
