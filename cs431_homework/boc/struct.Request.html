unsafe fn start_enqueue(&self, behavior: *const Behavior)

Start the first phase of the 2PL enqueue operation.

Enqueues self onto the target cown. Returns once all previous behaviors on this cown has finished enqueueing on all of its required cowns. This ensures the 2PL protocol.

§SAFETY

behavior must be a valid raw pointer to the behavior for self, and this should be the only enqueueing of this request and behavior.

Source

unsafe fn finish_enqueue(&self)

Finish the second phase of the 2PL enqueue operation.

Sets the scheduled flag so that subsequent behaviors can continue the 2PL enqueue.

§Safety

All enqueues for smaller requests on this cown must have been completed.

Source

unsafe fn release(&self)

Release the cown to the next behavior.

Called when self has been completed, and thus can allow the next waiting behavior to run. If there is no next behavior, then the cown’s tail pointer is set to null.

§Safety

self must have been actually completed.
